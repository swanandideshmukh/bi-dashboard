## 📌 Project Overview
A full-stack Business Intelligence Dashboard built as part of the Web Development Internship at **Codec Technologies**.

This platform allows businesses to visualize and analyze data through interactive charts, KPI monitoring, CSV data import, and exportable reports.

## ✨ Features
- 📈 Custom interactive dashboards
- 🎯 KPI monitoring (Revenue, Users, Orders)
- 📥 Data import from CSV files
- 📄 Exportable reports (CSV + Print)
- 📊 Bar, Line, Pie/Doughnut charts
- 🌙 Dark mode UI

## 🛠️ Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | React.js, React Router |
| Charts | Chart.js, React-Chartjs-2 |
| Backend | Node.js, Express.js |
| Database | PostgreSQL |
| Deployment | Vercel (Frontend), Render (Backend) |

## 🚀 Deployment
